# E航大微信小程序
E航大是一款校园服务查询类微信小程序，是碎片化、一站式、一体化校园移动门户，是适用于学生和教师的集课表、成绩查询、上网管理、空教室查询、体测、考试查询、素拓查询、洗衣机使用情况查询、图书馆借阅、拾卡信息、通知等等各功能于一身的服务查询类小程序。
